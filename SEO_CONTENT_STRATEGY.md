## üé® Content Format Guidelines

### Blog Post Structure
1. **Title** - Include primary keyword, compelling
2. **Meta Description** - 150-160 chars, include keyword, CTA
3. **Introduction** - Hook reader, state value proposition
4. **Table of Contents** - For long-form content
5. **Main Content** - Well-structured with H2/H3 headings
6. **Conclusion** - Summarize, include CTA
7. **Related Posts** - Internal linking

### SEO Best Practices
- **Keyword Density:** 1-2% (natural usage)
- **Word Count:** 1,500-3,000 words for guides
- **Internal Links:** 3-5 per post
- **External Links:** 2-3 authoritative sources
- **Images:** Alt text with keywords, optimized file sizes
- **Headings:** H1 (title), H2 (sections), H3 (subsections)
- **Meta Tags:** Title, description, OG tags
